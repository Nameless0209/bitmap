Сегментация бинарного изображения

Алгоритм проводит поиск и подсчет объектов(белых областей) на бинарном изображении, 
разметку объектов разными цветовыми кодами, и подсчет площади каждого из них.

На выходе получаем два файла Objects.txt и newImage.bmp.

Objects.txt  - Количество объектов и их площадь
newImage.bmp - Изображение полученное после разметки

loadImage() и saveImage() - Загрузка и сохранение изображения. В качестве параметра принимают путь к файлу.
generationColor() - Генерация нового цвета.
getObject() - Сохранение в файл Objects.txt найденных объектов.
processing() - Поиск объектов по вертикали.
objectSearch() - Проверка связей найденной части объекта
